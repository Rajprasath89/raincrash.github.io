Javascript Window

It would be great to find the dimensions of the window by Javascript and then designing your website for various dimensions. This would help in creating different versions of your website suitable for tablets and mobile. Different properties can be used to determine the size of the browser window (the browser viewport, NOT including toolbars and scrollbars). window.innerHeight - the inner height of the browser window window.innerWidth - the inner width of the browser window Here is a sample code that you can include in your script: 



var w=window.innerWidth
|| document.documentElement.clientWidth
|| document.body.clientWidth;

var h=window.innerHeight
|| document.documentElement.clientHeight
|| document.body.clientHeight;

This would let you know the browser window's height and width (NOT including toolbars/scrollbars).
